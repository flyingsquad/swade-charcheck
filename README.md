Check the character totals on a SWADE character.

A dialog is displayed with a list of the number of  Attributes, Skills, Edges, Hindrances and Advances. A point cost is calculated for each of these and compared to the number of points available to each type. The numbers of available points for each type are defined in the settings. The total price of gear the available currency are displayed at the bottom of the dialog.

As Traits are changed and items are added the totals update. When the costs equal the available points the message "Character is balanced" is displayed.

Attributes cost two points per die type. Edges cost two points each. Skills that are equal to or less than the linked Attribute are one point per die type, and two points if they exceed the linked attribute. Hindrances contribute 1 point (Minor) or 2 points (Major) to the total points available.

Advances contribute two points to the total, but note that no history is kept so if skills were earlier bought above the linked attribute, and then that linked attribute is raised later the totals won't be correct.

Messages will appear detailing any unsatisfied Edge requirements or weapons with unmet mininum Strength requirements.

Most "Other" type requirements cannot be evaluated properly, and will always be displayed. An exception is "Not": an Other requirement of the form "Not Ruthless" will search all the items on the character. If no item matches the text after "Not" the requirement is satisfied.

To open the character check dialog either:

* Clicking the Check icon in the character sheet title bar.
* Right-click the actor in the Actors tab and select Check Character Totals from the context menu.
